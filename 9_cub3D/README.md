# What is Cub3d about
Cub3d is a project about ray-casting and 3d graphics in C, where you must do a simple Wolfenstein clone using the minilib graphic library.
This project is about understanding how ray-casting works, managing user input, real time movement and working with a low level graphic library.
# Instructions how to compile and run Cub3d
To compile the program just run `make` or `make bonus` on the directory in the terminal and to run the program use `./cub3D maps/good/mapname.cub` or `./cub3D_bonus maps/bonus/good/mapname.cub`.

Only `.cub` map files can be used.
# Controls
 - `W/A/S/D` movement.
 - `Left arrow and Right arrow` to move the camera.
 - `M` to open and close the minimap.
 - `E` to open doors.
 - `R` to close doors.
 - `mouse` to move the camera.
 - `Q` to change camera mode (mouse mode or arrows mode).
 - `ESC` to exit the program
